# ThisPlay

This is a personal, open-source, offline-first progressive web application (PWA) built to track TV shows and movies using the TMDB API. 

Designed with a strict client-side architecture, it leverages `localForage` and Service Workers for seamless offline caching and data persistence. It is strictly for personal data management, has no commercial intent, and all user progress and metadata are stored and processed entirely locally on the client. Features include automated local zip backups and direct migration from TV Time.
